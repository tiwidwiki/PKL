# MoodDrink Landing Page

Simple static landing page for MoodDrink (toko online minuman). Files provided:

- `index.html` - Landing page with menu, order flow, modal checkout and product details.
- `styles.css` - Styling and simple animations.
- `script.js` - Frontend interactions for cart, modals, and subscription form.
- `admin.html` - Placeholder admin page.

## Menjalankan secara lokal

Jalankan server HTTP sederhana dari direktori `lamdingpage`:

```bash
cd /home/tiwiee/Documents/PKL/lamdingpage
python3 -m http.server 8000
```

Buka browser di `http://127.0.0.1:8000` dan buka `index.html`.
